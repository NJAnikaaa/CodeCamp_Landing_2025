## What is this project?

This is a website for CodeCamp 2025, which is a coding bootcamp. I made this website using HTML and CSS. It's a practice project where I learned how to build a complete website from scratch!

## What I learned

While making this project, I learned:
- How to write HTML code
- How to style websites with CSS
- How to make websites look good on phones and tablets (responsive design)
- How to add navigation bars
- How to create card layouts
- How to make contact forms

## Files in this project

- `index.html` - This is the main HTML file with all the content
- `style.css` - This file has all the styling (colors, fonts, layouts)
- `responsive-media-queries.css` - This makes the website work on mobile phones and tablets
- `images/` - Folder with all the images I used


## Things I used

- **HTML5** - For the structure
- **CSS3** - For styling
- **Font Awesome** - For the social media icons (Facebook, Instagram, Twitter)
- **Google Fonts** - I used Segoe UI font family

## Problems I faced (and solved!)

- At first, my cards were not aligning properly. I fixed it using flexbox!
- The website looked bad on mobile. I learned about media queries and fixed it!
- The navigation menu was tricky. I used a select dropdown to make it simple.
